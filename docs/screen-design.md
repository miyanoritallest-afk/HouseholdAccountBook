# 画面設計（画面一覧・画面遷移図・ワイヤーフレーム）

[← 要件定義書に戻る](./requirements.md)

---

## 1. 画面一覧

| # | 画面名 | URL | 説明 |
|---|--------|-----|------|
| P-1 | ログイン画面 | /login | メール・パスワードでログイン |
| P-2 | ユーザー登録画面 | /signup | 新規アカウント作成 |
| P-3 | ホーム（収支一覧）画面 | /home | 月別の収支一覧・月次集計を表示 |
| P-4 | 収支登録・編集画面 | /transactions/new, /transactions/:id/edit | 収支の新規登録・編集フォーム |
| P-5 | レポート画面 | /reports | グラフ・月次サマリーを表示 |
| P-6 | カテゴリ管理画面 | /categories | カテゴリの一覧・登録・編集・削除 |

---

## 2. 画面遷移図

```
[未ログイン状態]
  │
  ├─ /login          ログイン画面
  │    ├─ ログイン成功 ──────────────────→ /home（収支一覧）
  │    └─ 「新規登録はこちら」リンク ──→ /signup
  │
  └─ /signup         ユーザー登録画面
       └─ 登録成功 ────────────────────→ /home（収支一覧）

[ログイン済み状態]
  │
  ├─ /home           ホーム（収支一覧）画面  ← 起点
  │    ├─ 「＋追加」ボタン ──────────→ /transactions/new（収支登録）
  │    ├─ 収支行クリック ────────────→ /transactions/:id/edit（収支編集）
  │    ├─ ナビ「レポート」 ──────────→ /reports
  │    ├─ ナビ「カテゴリ」 ──────────→ /categories
  │    └─ ナビ「ログアウト」 ─────────→ /login（ログアウト後）
  │
  ├─ /transactions/new     収支登録画面
  │    ├─ 登録成功 ────────────────→ /home
  │    └─ キャンセル ──────────────→ /home
  │
  ├─ /transactions/:id/edit 収支編集画面
  │    ├─ 更新成功 ────────────────→ /home
  │    ├─ 削除成功 ────────────────→ /home
  │    └─ キャンセル ──────────────→ /home
  │
  ├─ /reports        レポート画面
  │    └─ ナビ「ホーム」 ───────────→ /home
  │
  └─ /categories     カテゴリ管理画面
       └─ ナビ「ホーム」 ───────────→ /home
```
