# 家計簿アプリ 要件定義書

## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| アプリ名 | 家計簿アプリ（Zaim風） |
| 目的 | 個人の収入・支出を記録・管理し、家計の状況を可視化する |
| 参考アプリ | Zaim |
| 作成日 | 2026-05-08 |

---

## 2. ターゲットユーザー

- 個人ユーザー（自分自身）
- 日々の収支を手軽に記録・管理したい人

---

## 3. 技術スタック

| レイヤー | 技術 |
|----------|------|
| フロントエンド | Next.js (React) |
| バックエンド | Ruby on Rails（API モード） |
| データベース | MySQL |
| 認証 | メール＆パスワード認証（devise 等） |

---

## 4. 対応デバイス

- PC ブラウザのみ（レスポンシブ対応は対象外）

---

## 5. 機能要件

### 5-1. 認証機能

| # | 機能 | 説明 |
|---|------|------|
| A-1 | ユーザー登録 | メールアドレス・パスワードで新規登録 |
| A-2 | ログイン | メールアドレス・パスワードでログイン |
| A-3 | ログアウト | セッションを終了する |

### 5-2. 収支管理機能

| # | 機能 | 説明 |
|---|------|------|
| B-1 | 収支登録 | 種別（収入／支出）・金額・カテゴリ・日付・メモを登録 |
| B-2 | 収支編集 | 登録済みの収支情報を編集 |
| B-3 | 収支削除 | 登録済みの収支情報を削除 |
| B-4 | 収支一覧表示 | 月別に収支の一覧を表示 |
| B-5 | 月次集計表示 | 選択月の収入合計・支出合計・残高を表示 |

### 5-3. カテゴリ管理機能

| # | 機能 | 説明 |
|---|------|------|
| C-1 | カテゴリ一覧 | 収入／支出それぞれのカテゴリを一覧表示 |
| C-2 | カテゴリ登録 | 新しいカテゴリを追加 |
| C-3 | カテゴリ編集 | カテゴリ名を編集 |
| C-4 | カテゴリ削除 | カテゴリを削除 |

**デフォルトカテゴリ（支出）**: 食費 / 交通費 / 娯楽費 / 日用品 / 医療費 / 光熱費 / 通信費 / その他  
**デフォルトカテゴリ（収入）**: 給与 / 副業 / その他

### 5-4. グラフ・レポート機能

| # | 機能 | 説明 |
|---|------|------|
| D-1 | 支出カテゴリ別円グラフ | 月別の支出をカテゴリ別に円グラフで表示 |
| D-2 | 月別収支棒グラフ | 過去6ヶ月の収入・支出・残高を棒グラフで表示 |
| D-3 | レポート画面 | グラフと月次集計をまとめたレポートページ |

---

## 6. 非機能要件

| 項目 | 内容 |
|------|------|
| セキュリティ | ログイン済みユーザーのみ自分のデータにアクセス可能 |
| データ保護 | 他ユーザーのデータは参照・操作不可 |
| パフォーマンス | 一覧・グラフ表示は3秒以内に表示 |
| ブラウザ対応 | Google Chrome 最新版 |

---

## 7. 画面一覧

| # | 画面名 | 説明 |
|---|--------|------|
| P-1 | ログイン画面 | メール・パスワードでログイン |
| P-2 | ユーザー登録画面 | 新規アカウント作成 |
| P-3 | ホーム（収支一覧）画面 | 月別の収支一覧・月次集計を表示 |
| P-4 | 収支登録・編集画面 | 収支の新規登録・編集フォーム |
| P-5 | レポート画面 | グラフ・月次サマリーを表示 |
| P-6 | カテゴリ管理画面 | カテゴリの一覧・登録・編集・削除 |

---

## 8. データベース設計（概要）

### users テーブル
| カラム名 | 型 | 説明 |
|----------|----|------|
| id | BIGINT | 主キー |
| email | VARCHAR | メールアドレス（ユニーク） |
| password_digest | VARCHAR | ハッシュ化パスワード |
| created_at | DATETIME | 作成日時 |
| updated_at | DATETIME | 更新日時 |

### categories テーブル
| カラム名 | 型 | 説明 |
|----------|----|------|
| id | BIGINT | 主キー |
| user_id | BIGINT | ユーザーID（外部キー） |
| name | VARCHAR | カテゴリ名 |
| category_type | ENUM | 種別（income / expense） |
| created_at | DATETIME | 作成日時 |
| updated_at | DATETIME | 更新日時 |

### transactions テーブル
| カラム名 | 型 | 説明 |
|----------|----|------|
| id | BIGINT | 主キー |
| user_id | BIGINT | ユーザーID（外部キー） |
| category_id | BIGINT | カテゴリID（外部キー） |
| transaction_type | ENUM | 種別（income / expense） |
| amount | DECIMAL | 金額 |
| date | DATE | 日付 |
| memo | TEXT | メモ（任意） |
| created_at | DATETIME | 作成日時 |
| updated_at | DATETIME | 更新日時 |

---

## 9. API エンドポイント（概要）

### 認証
| メソッド | パス | 説明 |
|----------|------|------|
| POST | /api/v1/auth/signup | ユーザー登録 |
| POST | /api/v1/auth/login | ログイン |
| DELETE | /api/v1/auth/logout | ログアウト |

### 収支
| メソッド | パス | 説明 |
|----------|------|------|
| GET | /api/v1/transactions | 収支一覧取得（月指定可） |
| POST | /api/v1/transactions | 収支登録 |
| PUT | /api/v1/transactions/:id | 収支更新 |
| DELETE | /api/v1/transactions/:id | 収支削除 |

### カテゴリ
| メソッド | パス | 説明 |
|----------|------|------|
| GET | /api/v1/categories | カテゴリ一覧取得 |
| POST | /api/v1/categories | カテゴリ登録 |
| PUT | /api/v1/categories/:id | カテゴリ更新 |
| DELETE | /api/v1/categories/:id | カテゴリ削除 |

### レポート
| メソッド | パス | 説明 |
|----------|------|------|
| GET | /api/v1/reports/monthly | 月次集計取得 |
| GET | /api/v1/reports/category_summary | カテゴリ別支出集計取得 |
